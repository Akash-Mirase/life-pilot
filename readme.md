# вљ™ Installation

## Clone Repository

```bash
git clone https://github.com/Akash-Mirase/life-pilot.git
```

```
cd life-pilot
```

---

## Backend

```
cd backend
npm install
```

Create `.env`

```
PORT=5000

MONGODB_URI=your_mongodb_uri

JWT_SECRET=your_secret

JWT_EXPIRES_IN=7d

GEMINI_API_KEY=your_gemini_api_key

NODE_ENV=development

FRONTEND_URL=http://localhost:5173
```

Run

```
npm run dev
```

---

## Frontend

```
cd frontend

npm install
```

Create `.env`

```
VITE_API_URL=http://localhost:5000/api
```

Run

```
npm run dev
```
